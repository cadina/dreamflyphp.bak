Please follow the guides below when adding your code into the framework:

Use english for identifier and comments.
Use UTF-8 encoding for code files.
Use spaces for indentation (4 spaces each indentation).
Add 'C' prefix before your class's names.
Add 'I' prefix before your interface's names.
Write interface in "system/interfaces.php".
Make your class extends the CBase class.
add global constants and functions into "bootstrap.php".

Improvements for this file are welcomed.

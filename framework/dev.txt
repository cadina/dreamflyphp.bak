Please follow the guides below before you contribute to the project:

Use english for identifier and comments.
Use UTF-8 encoding for code files.
Use spaces for indentation (4 spaces each indentation).
Add 'C' prefix before your class names.
Add 'I' prefix before your interface names.
Add 'T' prefix befoer your trait names;
add global constants and functions into "bootstrap.php".

Improvements for this file are welcomed.
